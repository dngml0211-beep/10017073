# K-RS 어휘 챌린지 (Vocabulary Challenge) 상세 기획

## 1. 개요
K-RS 어휘 챌린지는 아이들이 독서 외에도 기초 어휘력을 다질 수 있도록 설계된 **게이미피케이션 학습 콘텐츠**입니다. 연령별 발달 단계를 고려하여 유아(PreK) 단계와 초등 단계로 구분하여 운영됩니다.

---

## 2. 유아 (PreK) 단계 : 반응형 언어 놀이

### 2.1 기획 의도 및 콘셉트
* [cite_start]**벤치마킹:** 듀오링고 ABC 앱 (Duolingo ABC) [cite: 172]
* [cite_start]**핵심 콘셉트:** 유아에게 **질문하지 않고, 반응하게 만드는** 언어 경험 앱 [cite: 173]
* **주요 특징:**
    1.  [cite_start]**직관적 반응:** 질문을 던지는 방식이 아니라 듣고, 보고, 터치하며 즉각적으로 반응하도록 설계 [cite: 174]
    2.  [cite_start]**실패 없는 경험:** 점수나 실패 표시가 없으며, 틀리면 다시 경험하게 유도하여 성취감 보호 [cite: 175]
    3.  [cite_start]**단순한 UX:** 한 화면에 오직 한 가지 행동만 요구 [cite: 176]
    4.  [cite_start]**마이크로 러닝:** 미션당 5~10초, 전체 2~3분 안에 완료 가능한 짧은 호흡 [cite: 177]

### 2.2 학습 콘텐츠 및 플로우
* [cite_start]**테마:** '다람이와 도토리 수집 놀이' - 학습을 통해 도토리를 모으는 스토리텔링 적용 [cite: 178, 208]
* **활동 유형:**
    * [cite_start]**소리와 낱말 그림 매칭:** 청각 자극과 시각 이미지를 연결 [cite: 180]
    * [cite_start]**자소 조합:** 글자의 구성 원리 파악 [cite: 181]
    * [cite_start]**문장 속 낱말 찾기:** 문맥 인지 능력 향상 [cite: 197]
* **진행 플로우:**
    1.  [cite_start]**맵 화면:** 도토리를 모으러 가는 경로 표시, 잠금(Lock) 설정 등을 통해 단계별 진행 유도 [cite: 208]
    2.  [cite_start]**문항 진행:** 회차당 약 5문항 진행 [cite: 209]
    3.  [cite_start]**최종 완료:** 평가 결과 점수가 아닌 '성취 화면'으로 구성 (모은 도토리 수로 배지 보상 제공 가능) [cite: 210, 211, 213]

## 3. 초등 단계 : 어휘 학습 게임

### 3.1 기획 의도 및 콘셉트
* [cite_start]**콘셉트:** 학년별 기초 어휘 학습을 위한 **어휘 챌린지 게임** [cite: 216]
* [cite_start]**진도 정책:** 독서 표준 진도와는 별개로, **개인의 어휘 학습 속도에 따라** 진도 진행 [cite: 216]
* [cite_start]**학습 영역:** 어휘, 맞춤법, 속담, 관용구 등 포괄적 언어 지식 학습 [cite: 217]

### 3.2 퀴즈 유형 및 UI 구성
초등 단계는 흥미 유발을 위해 게임 요소를 적극 도입하여 힌트를 조합하거나 제한 시간 내에 푸는 방식으로 구성됩니다.

#### **① 세 가지 힌트 조합 퀴즈**
* [cite_start]주어진 3가지 힌트를 조합하여 정답 어휘를 유추하는 방식 [cite: 233]
* **예시 문제:**
    1.  감칠맛이 나게 달다.
    2.  흥미가 나게 아기자기하거나 간드러진 느낌이 있다.
    3.  편안하고 포근하다.
    * [cite_start]**정답:** 달콤하다 (UI 예시 화면 기반 유추) [cite: 222, 223]

#### **② 초성 힌트 퀴즈**
* [cite_start]어휘의 뜻 풀이와 함께 자음(초성) 힌트를 제공하여 정답 입력 [cite: 234]
* **예시 화면:**
    * [cite_start]책장 속 책 제목에 초성 **'ㄷㅋㅎㄷ'** 표시 [cite: 219]
    * [cite_start]책 제목 **'혈액형'** 제시 [cite: 220]

#### **③ 블록 맞추기 / 타임 어택**
* [cite_start]**블록형 키보드:** 화면 하단에 글자 블록(리, 언, 착, 의, 수, 말...)을 나열하여 정답 터치 유도 [cite: 232]
* [cite_start]**제한 시간:** "남은 시간 27초" 등 타임 어택 요소를 넣어 긴장감 부여 [cite: 228]
* **문제 예시:**
    * Q. 목적한 곳에 다다르다 (네 글자)
    * [cite_start]Q. 반의어 찾기 [cite: 232]

## 4. 운영 및 관리
* **배치:** K-RS 메인 화면 및 주차별 학습 플로우에 포함
* [cite_start]**빈도:** 주 1회 이상, 관리 수업 및 자기주도 학습 흐름 속에 배치 (1주, 3주차 코어북 읽기 후 / 2주, 4주차 이북 읽기 후 배치) [cite: 338, 361]
* **역할:** 독서 퀴즈(이해력 평가)와 별도로 어휘력 자체를 강화하는 보조 학습 수단으로 활용
