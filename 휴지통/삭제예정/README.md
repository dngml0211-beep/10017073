# 북클럽 3.0 프로토타입

## 📖 개요

북클럽 3.0은 **AI 독서버디와 함께하는 어린이 독서 지원 플랫폼**입니다. 독서 전·중·후 전 과정에서 아이가 스스로 선택하고 학습할 수 있도록 돕는 인터랙티브 독서 서비스입니다.

## 🎯 핵심 가치

- **독서 동반자형 AI 서비스**: 독서버디 "다람이"가 책 읽기, 퀴즈, 독후활동을 자연스럽게 연결
- **게이미피케이션**: 레벨 시스템, 배지, 진행률 표시로 독서 동기 부여
- **개인화 추천**: AI 기반 도서 추천 및 학습 경로 제공

## 🛠️ 기술 스택

### 프론트엔드
- **React 18**: 컴포넌트 기반 UI 구성
- **Babel Standalone**: 브라우저 내 JSX 변환
- **Vanilla CSS**: 반응형 디자인 구현

### 프로토타입 특징
- 단일 HTML 파일로 구현된 완전 동작 프로토타입
- CDN을 통한 React 라이브러리 로딩
- 외부 의존성 없이 브라우저에서 바로 실행 가능

## 📱 화면 구조

### 1. 홈 (메인화면)
- **독서버디 "다람이"** 캐릭터 인터랙션
- **코어북 섹션**: 권리여책 도서 목록
- **개인 추천 도서**: AI 기반 맞춤 추천 (2개 섹션)
- 책마다 진행률 표시 (프로그레스 바)

### 2. 라이브러리
- **레벨별 탭**: PreK-1, K 1-1, K 1-2, K 2-1, K 2-2
- **도서 그리드 뷰**: 책 표지 + 제목 + 진행률
- **AI 독서버디 추천**: 대화형 책 추천 기능
- 책 읽기 상태 관리 (읽는 중, 완료, 미독)

### 3. 마이페이지
- **프로필 영역**: 사용자 정보, 레벨 (책벌레 Level 3)
- **독서 통계**: 완료한 책, 진행 중인 책, 연속 독서 일수
- **획득 배지**: 독서 활동으로 획득한 배지 표시
- **독서 기록**: 캘린더 뷰로 독서 이력 확인

### 4. 독서 화면 (리더)
- **책 읽기 뷰어**: 페이지별 텍스트 표시
- **페이지 네비게이션**: 이전/다음 페이지 버튼
- **독서버디 지원**: 다람이 격려 메시지
- **액션 버튼**: 퀴즈 풀기, 독후활동 바로가기

### 5. 퀴즈 화면
- **진행률 표시**: 문제 번호 + 프로그레스 바 (예: 1/5)
- **객관식 문제**: 4지선다형 선택
- **독서버디 응원**: 힌트 및 격려 메시지
- **실시간 피드백**: 정답/오답 모달 표시

### 6. 독후활동 (리텔링)
- **음성/텍스트 입력**: 책 내용을 다람이에게 이야기하기
- **AI 피드백**: 독서 이해도 확인
- **활동 가이드**: 리텔링 방법 안내

### 7. AI 챗봇
- **독서버디 채팅**: 다람이와 대화
- **책 추천 대화**: AI가 대화를 통해 책 추천
- **독서 도움**: 책 읽기 관련 질문 응답

- 3열 그리드 레이아웃

- 4열 그리드 레이아웃
- 확장된 콘텐츠 영역
- 터치 최적화

### 데스크탑 (1200px)
- 6열 그리드 레이아웃 (홈), 5열 (라이브러리)
- 좌측 사이드바 네비게이션
- 하단 네비게이션 숨김
- 가로 레이아웃 최적화

## 🎨 디자인 시스템

### 컬러 팔레트
```css
Primary Colors:
- 메인 블루: #4A90E2 (신뢰, 학습)
- 서브 그린: #7ED321 (성장, 성공)
- 포인트 오렌지: #FF9800 (독서버디)

Secondary Colors:
- 배경 화이트: #FFFFFF
- 배경 그레이: #F5F5F5
- 텍스트: #333333
- 보더: #E0E0E0
```

### 타이포그래피
- **헤더**: 20px, Bold
- **섹션 제목**: 18px, Bold
- **본문**: 16px, Regular
- **캡션**: 12px, Regular
- **버튼**: 15px, Semi-Bold

### 애니메이션
- **페이지 전환**: 0.3s ease (fadeIn)
- **독서버디**: bounce 애니메이션 (무한 반복)
- **레벨업**: celebration 애니메이션
- **호버 효과**: transform, scale

## ✨ 주요 기능

### 1. 독서버디 "다람이" AI
- **책 읽기 지원**: 페이지 정지 시 격려 멘트
- **퀴즈 지원**: 무응답 시 힌트 제공
- **도서 추천**: 대화형 맞춤 추천
- **독서 독려**: 재방문 유도 메시지

### 2. 게이미피케이션
- **레벨 시스템**: 책벌레 Level 1~10
- **K-RT 레벨**: PreK-1, K 1-1, K 1-2, K 2-1, K 2-2
- **배지 획득**: 독서 활동 달성 시 배지 수여
- **진행률 표시**: 시각적 프로그레스 바

### 3. 독후활동
- **리텔링**: 음성/텍스트로 책 내용 이야기하기
- **퀴즈**: 이해도 확인 문제
- **영상 생성**: AI가 리텔링 기반 영상 생성 (구현 예정)

### 4. 통계 및 기록
- **독서 통계**: 완료 권수, 진행 중, 연속 일수
- **독서 기록**: 캘린더 뷰
- **학부모 대시보드**: 자녀 독서 현황 모니터링 (구현 예정)
