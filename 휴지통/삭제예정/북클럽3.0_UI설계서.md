# 북클럽 3.0 UI 설계서

## 1. 개요

### 1.1 서비스 컨셉
- **독서 동반자형 AI 서비스**: 독서 전·중·후 전 과정에서 아이가 스스로 선택할 수 있도록 돕는 AI 기반 독서 플랫폼
- **핵심 가치**: 읽기 흐름이 끊어지는 순간을 이어주고, 책-퀴즈-독후활동-재방문을 자연스럽게 연결

### 1.2 타겟 사용자
- 1~2학년: 오늘의 학습 화면 중심
- 3~6학년: 오늘의 학습 화면 중심
- 학부모: 학습 현황 및 성과 모니터링

---

## 2. 화면 구조 및 네비게이션

### 2.1 전체 화면 구조

```
북클럽 3.0
├── 메인화면 (홈)
│   ├── 코어북 (권리여책 도서)
│   ├── 개인 추천 도서 1
│   ├── 개인 추천 도서 2
│   ├── 북클럽 투데이 (큐레이션)
│   └── 독서버디 캐릭터 (다람이)
│
├── 라이브러리
│   ├── 레벨별 도서 (PreK-1, K 1-1, K 1-2...)
│   ├── 도서 검색
│   ├── AI 독서버디 도서 추천
│   └── 독서 기록 확인
│
├── 마이페이지
│   ├── 책벌레 레벨 현황
│   ├── 독서 기록
│   ├── 획득 배지
│   ├── 학습 통계
│   └── 설정
│
├── 독서 화면
│   ├── 책 읽기 뷰어
│   ├── 독서버디 지원 기능
│   ├── 퀴즈 콘텐츠

│
└── 스마트올 연계
    ├── 독후활동 (유레카 연계)
    ├── 누리집 독서
    └── 학습 콘텐츠 연계
```

### 2.2 주요 네비게이션
- **하단 탭 네비게이션** (4개)
  - 홈 (메인화면)
  - 라이브러리
  - 마이페이지
  - 스마트올 연계 (선택적)

#### 3.1.2 주요 요소
1. **독서버디 캐릭터 (다람이)**
   - 위치: 상단 중앙
   - 상태: 인사말, 독서 독려 멘트
   - 인터랙션: 탭하면 대화 모드 진입

2. **코어북 섹션**
   - 권리여책 도서 표시
   - 가로 스크롤 형태
   - 각 책마다 진행률 표시

3. **개인 추천 도서 섹션**
   - AI 기반 개인화 추천
   - 2개 섹션으로 구분 (테마별/레벨별)
   - 가로 스크롤 형태

4. **북클럽 투데이**
   - 큐레이션 콘텐츠 진입점
   - 일별 추천 도서 및 활동

   - 퀴즈 점수, 독후활동 완료 여부

### 3.4 독서 화면 (책 읽기)

#### 3.4.1 화면 구성
```
┌─────────────────────────────┐
│ [←] 책 제목            [⋮]  │
├─────────────────────────────┤
│                             │
│                             │
│     책 내용 영역            │
│     (전체 화면)             │
│                             │
│     텍스트 + 이미지          │
│                             │
│                             │
│                             │
│                             │
│  ┌───────┐                  │
│  │다람이 │ "좋아요! 계속    │
│  │ 버디  │  읽어볼까요?"    │
│  └───────┘                  │
│                             │
├─────────────────────────────┤
│ [◀] 페이지 3/20 [▶]         │
│ [퀴즈] [독후활동] [저장]    │
└─────────────────────────────┘
```

#### 3.4.2 독서버디 지원 기능

**1) 책 읽기 지원**
- **트리거**: 페이지 정지 15~20초, 집중도 저하, 중간 종료
- **동작**: 동참하여 맨트 제공
- **멘트 예시**:
  - "잠깐 쉬어가도 괜찮아요!"
  - "조금만 더 읽어볼까요?"
  - "재미있는 부분이 나올 거예요!"

**2) 퀴즈 콘텐츠 지원**
- **트리거**:
  - 퀴즈 시작점검/로제 10초 무응답
  - 연속 오답 등
- **동작**: 맨트 제공으로 민감하지 않게 연속 동참 유도
- **멘트 예시**:
  - "힌트가 필요하신가요?"
  - "천천히 생각해봐요!"

**3) 중간 독후 활동 지원**
- **기능**: 리텔링, 서술형 활동의 캐릭터/스토리 제공
- **동작**:
  - 책을 읽고 나서 실시 아이템 이야기
  - 독서 버디를 메인으로 세워 이지 버디에게 버튼 메시지 남기는 간단 활동 시작 딜기능 박스화 제공 추가
- **예시**: "선덕여왕" 책 읽고 다람이에게 주요 내용 이야기하면 영상 생성

**4) 라이브러리 도서 추천**
- **트리거**: 카테고리 버튼 누를 때
- **동작**: AI 독서 버디가 대화를 통해 책 추천
- **멘트 예시**:
  - "오늘은 어떤 이야기가 궁금해요?"
  - "모험 이야기와 동화 중 뭐가 좋을까요?"

**5) 일별/재방문 독려**
- **트리거**:
  - 하루 미접속
  - 또는 일시 중지 이후 재방문 촉발
  - 책 읽기나 퀴즈/독후활동 중단
- **동작**: 가이드 맨트 제공
- **멘트 예시**:
  - "오랜만이에요! 어제 읽던 책 계속 읽을까요?"
  - "이야기가 궁금하지 않아요?"

4. **독서버디 응원**: 무응답 10초 후 힌트 제공
5. **결과 피드백**: 정답/오답 즉시 표시

## 4. UI/UX 가이드라인

### 4.1 디자인 원칙

1. **어린이 친화적 디자인**
   - 큰 버튼과 아이콘
   - 명확한 시각적 피드백
   - 직관적인 인터페이스

2. **독서버디(다람이) 일관성**
   - 모든 화면에서 캐릭터 노출
   - 일관된 톤앤매너
   - 긍정적이고 격려하는 메시지

3. **가독성 우선**
   - 적절한 폰트 크기 (최소 14pt)
   - 충분한 줄 간격
   - 고대비 색상 조합

4. **게이미피케이션 요소**
   - 레벨 시스템
   - 배지 및 보상
   - 진행률 시각화

### 4.2 컬러 시스템

```
Primary Colors:
- 메인 블루: #4A90E2 (신뢰, 학습)
- 서브 그린: #7ED321 (성장, 성공)
- 포인트 옐로우: #F5A623 (활력, 재미)

Secondary Colors:
- 배경 화이트: #FFFFFF
- 텍스트 블랙: #333333
- 그레이: #8E8E93
- 경고 레드: #FF3B30

Gradient:
- 레벨업: #4A90E2 → #7ED321
- 배지: #F5A623 → #FF3B30
```

### 4.3 타이포그래피

```
Heading 1: 24pt, Bold
Heading 2: 20pt, Bold
Heading 3: 18pt, Semi-Bold
Body: 16pt, Regular
Caption: 14pt, Regular
Button: 16pt, Semi-Bold
```

### 4.4 아이콘 스타일
- 라인 스타일 아이콘
- 2px 두께
- 라운드 모서리
- 단색 또는 2색 조합

### 4.5 애니메이션
- 페이지 전환: 0.3s ease
- 버튼 탭: 0.2s bounce
- 독서버디 등장: 0.4s slide-in
- 레벨업 효과: 1s celebration

## 5. 주요 인터랙션 시나리오

### 5.1 신규 사용자 온보딩

```
Step 1: 웰컴 스크린
"안녕! 나는 독서버디 다람이야!"

Step 2: 레벨 테스트
"먼저 네 독서 레벨을 확인해볼까?"

Step 3: 관심사 선택
"어떤 이야기를 좋아해?"
[모험] [동화] [과학] [역사]

Step 4: 첫 책 추천
"이 책은 어떨까?"
[책 표지 + 설명]

Step 5: 독서 시작
"그럼 함께 읽어볼까? 시작!"
```

### 5.2 책 읽기 플로우

```
1. 책 선택
   → 책 상세 페이지 진입

2. [읽기 시작] 버튼 탭
   → 뷰어 화면 로딩

3. 페이지별 읽기
   → 좌우 스와이프 또는 버튼으로 페이지 이동

4. 중간 저장
   → 자동 저장 + 수동 북마크 가능

5. 읽기 완료
   → 축하 메시지
   → 퀴즈/독후활동 제안
```

### 5.3 독서버디 AI 추천 플로우

```
1. [AI 독서버디 추천받기] 버튼 탭

2. 다람이 등장
   "안녕! 오늘은 어떤 책을 찾고 있어?"

3. 사용자 응답 (음성 또는 텍스트)
   예: "모험 이야기요!"

4. 다람이 추천
   "그럼 이 책들은 어때?"
   [책 3권 제시]

5. 사용자 선택
   → 책 상세 페이지로 이동

6. 피드백 수집
   "이 추천이 마음에 들었어?"
   [👍 좋아요] [👎 아니요]
```

### 5.4 레벨업 플로우

```
1. 책 읽기/퀴즈 완료
   → 경험치 획득

2. 레벨업 조건 달성
   → 축하 애니메이션 실행

3. 레벨업 화면
   ┌─────────────────────┐
   │  🎉 레벨업!         │
   │                     │
   │  Level 3 → Level 4  │
   │  K 1-1 → K 1-2      │
   │                     │
   │  새로운 책들이      │
   │  해금되었어요!      │
   │                     │
   │  [확인]             │
   └─────────────────────┘

4. 배지 획득
   → 마이페이지에 자동 추가

5. 새 도서 접근 권한
   → 라이브러리 업데이트
```
